# Die Klugscheißer-Bibliothek

Wissen für den Blauen Elefanten. Törööö... wusstest du eigentlich...?

Jedes Element hat seine Geschichte. Der Elefant kennt sie alle.

---

## ⚙️ Metall

**Ada Lovelace (1843)**
- Schrieb das erste Computerprogramm — für eine Maschine die es noch nicht gab
- Die "Analytical Engine" von Charles Babbage existierte nur auf Papier
- Ada schrieb trotzdem Anweisungen dafür. DAS ist Programmieren.
- Sie sagte voraus: Maschinen könnten eines Tages Musik komponieren (180 Jahre vor Spotify)

**al-Khwarizmi (ca. 780–850)**
- Persischer Mathematiker in Bagdad
- Sein Name wurde zu "Algorithmus" — das Wort für jede Rechenanweisung
- Schrieb das Buch "al-Kitab al-mukhtasar fi hisab al-jabr wal-muqabala"
- Daraus wurde "Algebra". Ein Wort, zwei Fächer.

**Alan Turing (1936)**
- Erfand die "Turing-Maschine" — ein Gedankenexperiment
- Bewies: EINE Maschine kann ALLES berechnen was berechenbar ist
- Knackte im 2. Weltkrieg den Enigma-Code und rettete Millionen Leben
- Vater der Künstlichen Intelligenz

## 🪵 Holz

**Der Abakus (ca. 2400 v. Chr.)**
- Erster "Computer" — Kugeln auf Stäben
- Wurde in Mesopotamien, China, Rom und Japan benutzt
- Ein geübter Abakus-Rechner ist SCHNELLER als ein Taschenrechner
- Prinzip: Position bestimmt den Wert (wie Einer, Zehner, Hunderter)

**Jacquard-Webstuhl (1804)**
- Joseph-Marie Jacquard baute einen Webstuhl der mit Lochkarten gesteuert wurde
- Lochkarte = Programm. Loch = 1, kein Loch = 0. Binär!
- Ada Lovelace sah die Lochkarten und hatte die Idee: Maschinen können RECHNEN, nicht nur weben
- Von Lochkarten zu USB-Sticks: 220 Jahre, gleiches Prinzip

## 🔥 Feuer

**Gottfried Wilhelm Leibniz (1679)**
- Erfand das Binärsystem: nur 0 und 1
- Alles was ein Computer tut basiert auf seiner Idee
- Auch: Erfinder der ersten mechanischen Rechenmaschine die multiplizieren konnte
- Unabhängig von Newton die Infinitesimalrechnung entwickelt (Streit bis heute)

**George Boole (1847)**
- Erfand die Boolesche Logik: WAHR oder FALSCH, UND, ODER, NICHT
- Jede if-Abfrage in jedem Programm der Welt basiert auf Boole
- `if (feuer === true)` — das ist George Boole in einer Zeile
- Sein Buch: "The Mathematical Analysis of Logic" — 23 Seiten die die Welt veränderten

**Claude Shannon (1937)**
- Zeigte: Boolesche Logik lässt sich mit elektrischen Schaltern bauen
- Schalter AN = 1 = WAHR, Schalter AUS = 0 = FALSCH
- Damit war klar: Computer sind möglich. Nicht irgendwann. JETZT.
- Seine Masterarbeit gilt als die wichtigste Masterarbeit aller Zeiten

## 🌊 Wasser

**Java (die echte Insel)**
- Insel in Indonesien, berühmt für Kaffee
- Die Programmiersprache Java wurde 1995 in einem Café erfunden
- James Gosling trank Java-Kaffee und nannte die Sprache danach
- "Write once, run anywhere" — auf jedem Computer der Welt

**Grace Hopper (1952)**
- Erfand den ersten Compiler: ein Programm das menschliche Sprache in Maschinencode übersetzt
- Fand den ersten echten "Bug": eine Motte die im Computer feststeckte
- "Es ist einfacher um Vergebung zu bitten als um Erlaubnis" — ihr Motto
- Programmierte den UNIVAC I, einen der ersten kommerziellen Computer

**Makro der Hai (Informatik)**
- Ein Makro ist ein kleines Stück Code das sich RIESIG aufbläht
- Sieht harmlos aus, wird aber beim Kompilieren zu viel mehr Code
- Deswegen warnen Programmierer: "Vorsicht vor Makros!"
- In C und C++ berüchtigt für schwer findbare Fehler

## 🟫 Erde

**Babylon (ca. 2000 v. Chr.)**
- Babylonische Mathematiker rechneten im 60er-System (Sexagesimal)
- Deswegen hat eine Stunde 60 Minuten und ein Kreis 360 Grad
- Sie hatten Tabellen für Quadratzahlen, Kubikzahlen, Wurzeln
- Erste bekannte Algorithmen: Schritt-für-Schritt-Anleitungen auf Tontafeln

**Pythagoras (ca. 570–495 v. Chr.)**
- "Alles ist Zahl" — Grundüberzeugung der Pythagoreer
- Isopsephie: Α=1, Β=2 ... Ω=800. Jedes Wort hat einen Zahlenwert.
- Entdeckte die Pentatonik: 5 Töne aus reinen Quintverhältnissen
- Die 5 Elemente im Spiel klingen nach seiner Entdeckung (C D E G A)

**Euklid (ca. 300 v. Chr.)**
- Sein Algorithmus für den größten gemeinsamen Teiler funktioniert seit 2300 Jahren
- Ältester Algorithmus der noch heute in Computern läuft
- Bewies: Es gibt unendlich viele Primzahlen (in 3 Zeilen!)
- Sein Buch "Elemente" war 2000 Jahre lang DAS Mathematik-Lehrbuch

**ASCII (1963)**
- American Standard Code for Information Interchange
- A=65, B=66, C=67 ... Z=90, a=97, 0=48, Leerzeichen=32
- 128 Zeichen reichten für Englisch. Für Deutsch nicht (ä, ö, ü!)
- Deswegen gibt es heute Unicode: 149.813 Zeichen aus allen Sprachen der Welt

**Gematria (Hebräisch) / Isopsephie (Griechisch)**
- Buchstaben = Zahlen. Worte = Summen. Gleiche Summe = mystische Verbindung.
- Das älteste "Encoding" der Welt
- Direkte Vorfahren von ASCII und Unicode
- Pythagoras, die Kabbala und moderne Computer: gleiche Grundidee

## Querverbindungen (für den Elefanten)

- **Leibniz' Binär → Booles Logik → Shannons Schalter → dein Computer**: Drei Männer, 258 Jahre, eine Idee.
- **Pythagoras → Isopsephie → ASCII → Unicode → Emojis**: Buchstaben waren IMMER Zahlen. Jetzt sind sie auch Bilder.
- **Abakus → Pascals Rechner → Babbages Engine → Adas Programm → dein Spiel**: Der erste Computer war ein Holzrahmen mit Kugeln.
- **al-Khwarizmi → "Algorithmus" → jedes Rezept in der Werkbank**: Wenn du Erde + Feuer zu Stein machst, folgst du einem Algorithmus aus dem 9. Jahrhundert.
- **"Außer Text nix gehext"**: Pythagoras wusste es. Ada wusste es. ASCII bewies es. Dein Spiel zeigt es.
